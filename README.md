# 🚀 Flask URL Shortener

A sleek and functional web application built to convert long, messy URLs into clean, short links. This project demonstrates full-stack development skills, focusing on backend logic and database management.

### 🛠️ Tech Stack
* **Backend:** Python (Flask Framework)
* **Database:** SQLite3 (For persistent data storage)
* **Frontend:** HTML5 & CSS3 (Modern, Responsive UI)
* **Logic:** Custom 6-character unique code generation

### ✨ Key Features
* **Instant Redirection:** Automatically routes users from the short link to the original destination.
* **Persistent Storage:** Uses SQLite to ensure links stay active even after server restarts.
* **Clean UI:** A minimalist, user-friendly dashboard for a seamless experience.
